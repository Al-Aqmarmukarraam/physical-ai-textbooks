## Overview

This repository contains a comprehensive textbook on Physical AI and Humanoid Robotics, built with Docusaurus. The textbook covers 6 chapters with 3 lessons each, focusing on the intersection of artificial intelligence and physical systems.

## Chapters

1. **Introduction to Physical AI** - Understanding the fundamentals of AI systems that interact with the physical world
2. **Basics of Humanoid Robotics** - Exploring the design and control of human-like robots
3. **ROS 2 Fundamentals** - Learning the Robot Operating System for robotics development
4. **Digital Twin Simulation** - Mastering simulation tools like Gazebo and Isaac Sim
5. **Vision-Language-Action Systems** - Integrating perception, language understanding, and physical action
6. **Capstone: Simple AI-Robot Pipeline** - Building and deploying complete AI-robot systems

## Features

- Complete textbook with 18 lessons following a consistent format
- AI-Native design with integrated RAG system for Q&A
- Clean, modern Docusaurus interface optimized for learning
- GitHub Pages deployment ready
- RAG-optimized content structure for vector embeddings

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run start`
4. Open [http://localhost:3000](http://localhost:3000) to view the textbook

## Building

To build the static site for deployment:

```bash
npm run build
```

The output will be in the `build` directory.

## Deployment

The site is configured for GitHub Pages deployment. Use the deploy script:

```bash
npm run deploy
```

## Contributing

This textbook is designed to be easily extensible. Each lesson follows a consistent format with frontmatter, introduction, core concept, example, and key takeaways.
